## Key Insight
Overall Grade: **C+** — Good bones, poor ergonomics. Outfit + IBM Plex Mono is a powerful luxury-tech pairing. Courier Prime as global body font undermines the luxury positioning. High-end real estate and DeFi both demand credibility through refinement.

## Recommended Type Scale (Major-Third, 1.25×)
- `display-2xl`: 96px / 0.92em / -0.03em
- `display-xl`: 80px / 0.94em / -0.02em
- `display-lg`: 64px / 0.95em / -0.02em
- `display-md`: 48px / 1.0em / -0.01em
- `body-lg`: 18px / 1.6em (new — for hero subtitles & lead paragraphs)
- `body`: 16px / 1.65em
- `body-sm`: 14px / 1.6em
- `label`: 12px / 1.4em / 0.08em

## Font Pairing Recommendation
- **Headlines + UI**: Outfit (keep)
- **Body text**: Source Serif 4 or Inter
- **Terminal / Code**: IBM Plex Mono (keep)
- **Accent / Manifesto**: Courier Prime — retain for short "system voice" moments only
